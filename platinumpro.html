Bagaimana cara Register OikuKasir?

Untuk pertama kali menggunakan aplikasi, anda bisa Register menggunakan data pengguna di bawah ini :

  • Username : email yang sudah di register
  • Password : pasword yang di masukan saat ergister akun admin di aplikasi kasir
Jika sudah berhasil Register, jangan lupa untuk mencoba login dengan email dan password yang sudah di register.

Bagaimana cara login OikuKasir?

Untuk pertama kali menggunakan aplikasi, akan ada dua halaman login :
1. Halaman login khusus admin terdapat keterangan Admin di bagian atas, seperti gambar berikut ini

  • Untuk masuk ke halaman login admin bisa di klik tombol login sebagai admin di bagian bawah halaman login user

    • Username : email yang sudah di register
    • Password : pasword yang di masukan saat register akun admin di aplikasi kasir
    2. Halaman login khusus karyawan atau user, seperti gambar berikut ini

    • No HP : 999999
    • Password : 1234
    Jika sudah berhasil login, akan masuk pada menu utama aplikasi.

    Pengaturan Toko dan Logo Toko

    Silhakan ubah nama toko, alamat, info kontak, PPN, jenis mata uang, sesuai dengan toko anda, nama toko ini akan digunakan ketika mencetak nota penjualan.
    Pada menu utama buka menu Pengaturan -> Informasi Toko, tombol paling atas.


    Bagaimana membuat kategori baru?

    Jangan lupa sesuaikan kategori produk berdasarkan kategori atau pengelompokan barang di toko anda, untuk saat ini sudah ada beberapa sample data yang kami sediakan.

    • Nama Kategori : Nama kelompok kategori produk yang anda jual.

    Bagaimana membuat produk baru?

    Jangan lupa sesuaikan produk dengan kebutuhan toko anda, untuk saat ini sudah ada beberapa sample data yang kami sediakan.

    1. Buka menu Produk > klik icon (+).
    2. Nama Produk : Nama produk yang di jual di toko.
    3. Kode Produk : Kode produk yang anda jual, harus unik dan tidak boleh sama dengan kode produk barang lain. Misal : P1111, P2222, P3333.
    4. Kategori : Pilih kategori produk sesuai dengan kategori yang sudah di buat serta pilih sesuaikan dengan kelompok jenis produk yang akan di masukan.
    5. Harga Modal/Beli : Harga modal atau harga beli atau harga kulak produk.
    6. Stok : Jumlah stok produk.
    7. Harga Jual : Harga jual produk (Harga Jual Ecer).
    8. Harga Jual Grosir : Harga jual produk Grosir.
    9. Satuan produk : Nama satuan jenis satuan produk.
    10. Berat Produk : ukuran berat per satuan produk.
    11. Nama Pemasok : Di isi nama pemasok atau suplier produk.
    12. Check List Variasi : Untuk 1 produk maksimal dapat memiliki 4 varian, serta dalam 1 varian bisa memiliki harga jual sendiri, stok sendiri, dan nama varian sendiri.
    13. Photo produk : Photo produk yang anda jual.
    14. Setelah semuanya terisi klik tombol tambah produk baru

    Kelola Stok Produk Barang

    Login sebagai admin melalui halaman login admin

    1. Masuk di menu produk
    2. Pilih atau cari produk yang akan ditambahkan jumlah stok nya
    3. klik edit produk
    4. isi jumlah produk


    Kelola Harga Produk Barang atau Edit Produk

    Login sebagai admin melalui halaman login admin

    1. Masuk di menu produk
    2. Pilih atau cari produk yang akan di edit
    3. klik edit produk
    4. isi perubahan yang di perlukan misal : harga jual, hrga modal, stok, dll


    Bagaimana membuat data pelanggan baru?

    Jika anda akan memulai melakukan transaksi penjualan, maka anda wajib membuat data pelanggan jika memang belum ada, data pelanggan ini nantinya akan di tampilkan ketika transaksi penjualan pada menu POS/Penjualan.

    1. Buka menu pelanggan > klik icon (+).
    2. Isi data pelanggan.
    3. Klik tambah pelanggan untuk menyimpan data..

    Bagaimana membuat data pemasok baru?

    Jika anda akan memulai melakukan penambahan produk, maka anda bisa membuat data pelanggan jika memang belum ada

    1. Buka menu supplier/pemasok > klik icon (+).
    2. Isi data pemasok.
    3. Klik tambah pemasok untuk menyimpan data..

    Bagaimana menambah stok lewat menu inventory dan melihat valuasi stok lewat inventory ?

    Jika anda akan menambahkan stok produk, bisa juga melalui fitur inventory serta jika ingin melihat valuasi saham gudang toko juga bisa melalui menu inventory ini

    1. Buka menu inventory
    2. Pada masing masing produk akan tertera daftar valuasi saham gudang
      • Hitungan valuasi total modal untuk masing masing produk
      • Hitungan estimasi valuasi total penjualan untuk masing masing produk
    3. Untuk menambah stok bisa klik salah satu produk yang akan di tambahkan stok nya
    4. Isi jumlah pada kolom kuantitas penambahan
    5. Klik tombol update

    Bagaimana menambah stok melalui proses proses wholesale/kulakan ?

    Jika anda akan menambahkan stok produk, bisa jga melalui fitur wholesale/kulakan

    1. Buka menu inventory
    2. Untuk menambah stok bisa klik salah satu produk yang akan di tambahkan stok nya
    3. Isi jumlah pada kolom kuantitas penambahan
    4. Klik cheklist tambahkan ke daftar pengeluaran
    5. Isi nama pengeluaran
    6. Catatan jika ada
    7. Jumlah pembayaran atau jumlah pengeluaran untuk kulakan
    8. Tanggal dan waktu pengeluaran

    Bagaimana cara membuat label harga dengan barcode atau QR ?

    Jika anda membuat label harga dengan tampilan barcode atau QR code yang bisa di tempelkan di kemasan produk

    1. Buka menu Label
    2. Pilih produk yang akan dicetak label nya
    3. Isi jumlah copy cetak dalam satu kali print Rekomendasi tidak lebih dari 20 copy
    4. Pilih jenis tampilan barcode atau QR code
    5. Klik Print Label

    Bagaimana menghubungkan printer via Bluetooth?

    Sebelum memulai transaksi, untuk kebutuhan cetak struk dan label jangan lupa hubungkan printer POS anda dengan bluetooth hp kasir yang digunakan.

    1. Enable bluetooth di android anda.
    2. Masuk ke pengaturan bluetooh HP kasir, cari dan pair devices printer anda.
    3. Jika sudah terhubung, langsung aja lanjut ke tahap transaksi.
    4. Pada menu order atau daftar pesanan/transaksi yang sudah dilakukan, pilih salah satu transaksi yang akan di cetak notanya.
    5. Setelah masuk ke detail pesanan klik tombol Print via Thermal printer di bagian bawah
    6. Pilih dan klik nama printer yang sudah di hubungkan misal RPP02

    Jika sewaktu-waktu, mengalami masalah Bluetooth sudah terhubung, namun tidak bisa cetak struk, coba lakukan unpair printer, kemudian hubungkan kembali ya.


    Bagaimana memulai transaksi penjualan?

    Sebelum membuat transaksi penjualan, anda harus sudah membuat data produk dan data pelanggan ya.

    1. Buka menu POS pada halaman utama aplikasi.
    2. Untuk merubah tampilah daftar produk bisa di klik tombol List atau Both
    3. Pilih produk yang dengan cara klik "add to cart".
    4. Jika produk anda memiliki varian maka otomatis akan muncul dialog untuk pemilihan varian dari produk induk yang tersedia
    5. Pilih nama varian yang akan di jual cengan klik pada checklist di samping nama varian produknya
    6. Untuk produk tanpa varian dialog ini tidak muncul, dan bisa di klik tombol "add to cart" secara berulang sesuai dengan jumlah produk yang di jual
    7. Untuk melanjutkan proses transaksi, klik icon keranjang belanja di pojok kanan atas.
    8. Selanjutnya akan tampil list produk yang di beli dan total harganya.
    9. Jangan lupa sesuiakan qty produk yang di transaksikan, dengan klik tombol +/-
    10. Klik "Proses order" untuk melanjutkan proses pembayaran.
    11. Order Type : PICKUP atau Non PICKUP, hanya informasi saja.
    12. Metode Pembayaran : CASH atau eWallet (hanya informasi saja).
    13. Grand total : Total penjualan yang sudah di kurangi diskon jika ada.
    14. Jangan lupa isi nominal pembayaran dari pembeli
    15. Jika ingin memberi diskon, jangan lupa input nilai diskon di kolom yang di sediakan ya.
    16. Klik PROSES, transaksi akan di proses. Selanjutnya anda akan di bawa ke halaman order list.
    17. Klik data order yang paling atas (terbaru), maka akan tampil details order.
    18. Nota Digital : untuk membuat nota digital yang bisa dikirim lewat whatsapp.
    19. Print Via Thermal Printer : untuk mencetak nota transaksi lewat printer dan di berikan ke pelanggan.

    produk akan otomatis mengurangi stok setelah proses transaksi dilakukan.


    Bagaimana menggunakan scanner external atau camera?

    Sambungkan dahulu scanner melalui sambunga USB OTG ke socket hp.

    1. Buka menu POS pada halaman utama aplikasi.
    2. Pilih icon gambar scanner external atau scanner camera pada ujung kanan atas di menu POS
    3. Pada dialog harga yang digunakan saat scan dan penjualan pilih salah satu bisa menggunakan hrga jual ecer atau grosir
    4. Mulai scan produk, nanti secara otomatis produk akan masuk ke daftar cart yang dijual

    produk akan otomatis mengurangi stok setelah proses transaksi dilakukan.


    Bagaimana transaksi penjualan GROSIR ?

    Sebelum membuat transaksi penjualan, anda harus sudah membuat data produk yang tersedia harga grosir dan data pelanggan ya, konsep penjualan grosirnya dengan memilih harga jual grosir terlebih dahulu kemudian pada menu cart bisa di atur jumlah penjualan produknya.

    Note : Untuk satu produk yang sama tidak bisa menggunakan harga ecer dan grosir sekaligus dalam 1 transaksi yang sama karena akan, mengakibatkan perhitungan yang tidak akurat pada laporan panjualan

    1. Buka menu POS pada halaman utama aplikasi.
    2. Untuk merubah tampilah daftar produk bisa di klik tombol List atau Both
    3. Pilih produk yang dengan cara klik "Jual Grosir" (Tombol Warna Oren) di bagian atas foto produk.
    4. Untuk melanjutkan proses transaksi, klik icon keranjang belanja di pojok kanan atas.
    5. Selanjutnya akan tampil list produk yang di beli dan total harganya pada menu cart.
    6. Jangan lupa sesuiakan qty produk yang di transaksikan, dengan klik tombol +/-
    7. Klik "Proses order" untuk melanjutkan proses pembayaran.
    8. Order Type : PICKUP atau Non PICKUP, hanya informasi saja.
    9. Metode Pembayaran : CASH atau eWallet (hanya informasi saja).
    10. Grand total : Total penjualan yang sudah di kurangi diskon jika ada.
    11. Jangan lupa isi nominal pembayaran dari pembeli
    12. Jika ingin memberi diskon, jangan lupa input nilai diskon di kolom yang di sediakan ya.
    13. Klik PROSES, transaksi akan di proses. Selanjutnya anda akan di bawa ke halaman order list.
    14. Klik data order yang paling atas (terbaru), maka akan tampil details order.
    15. Nota Digital : untuk membuat nota digital yang bisa dikirim lewat whatsapp.
    16. Print Via Thermal Printer : untuk mencetak nota transaksi lewat printer dan di berikan ke pelanggan.

    produk akan otomatis mengurangi stok setelah proses transaksi dilakukan.


    Bagaimana menghapus order / transaksi yang sudah dilakukan ?

    Secara default pengaturan aplikasi, hanya hak akses admin atau owner saja yang bisa menghapus transaksi yang sudah dilakukan.

    1. Buka menu pesanan/order.
    2. Pilih salah satu transaksi atau orderan yang akandi hapus.
    3. Klik pada bagian tombol delete kecil pada salah satu transaksi yang akan dihapus.
    4. Klik Yes

    Selamat, order yg sudah selesai sekarang menjadi bisa di hapus.

    Orderan yabg di hapus ini tidak akan dihitung pada laporan penjualan


    Bagaimana membuat transaksi pengeluaran atau beban usaha?

    • Buka menu pengeluaran > klik icon (+)
    • Masukan data pengeluaran atau beban usaha, klik "Tambah Pengeluaran".

    Melihat laporan transaksi penjualan dan pengeluaran

    Untuk versi OikuKasir - Grosir menyediakan 5 laporan untuk kebutuhan bisnis anda, di antaranya :

    1. Penjualan
    2. Pengeluaran
    3. Grafik Penjualan Bulanan
    4. Grafik Pengeluaran Bulanan
    5. Laba/Rugi

    • Total Pengeluaran : Total beban usaha selama periode yang di tentukan (semua pengeluaran, harian ini, bulanan ini, tahunan ini)

    • Total Penjualan : Total transaksi penjualan selama periode yang di tentukan (berdasarkan filter rentang tanggal transaksi, semua pengeluaran, harian ini, bulanan ini, tahunan ini)

    • Total Pendapatan/Keuntungan : Total transaksi penjualan yang sudah di akui sebagi pendapatan, dimana uang nya sudah kita terima. Total pendapatan harus = total penjualan, jika total pendapatan lebih kecil dari total penjualan maka ada transaksi yang belum lunas.

    • Laporan Laba Rugi Laporan untuk mengetahui pendapatan atau kerugian usaha anda. Pada laporan laba/rugi, terdapat 2 komponen, yaitu total pengeluaran, pendapatan/keuntungan.

    Backup & Restore Data

    Jika anda ingin mengganti perangkat namun tidak mau kehilangan data saat ini, cukup lakukan backup pada perangkat lama kemudian lakukan restore data di perangkat baru, berikut ini adalah caranya.

    1. Buka menu pengaturan, klik menu pencadangan data
    2. Klik Database Backup, lalu input nama file backup nya, misal "data toko". Klik Ya.
    3. Selanjutnya, buka aplikasi file manager, cari nama file dan folder tempat menyimpan data backup, pada contoh ini file nya bernama "data toko", maka cari file yang bermana "data toko" tersebut di peyimpanan hp
    4. Share atau kirim file backup yang sudah kita buat tadi ke perangkat baru.
    5. Taruh file hasil backup (pada contoh ini bernama "data toko") tersebut di dalam penyimpanan internal pada perangkat HP barunya ya.
    6. Buka aplikasi OikuKasir Grosir di perangkat baru
    7. Masuk menu pengaturan -> Pencadangan data -> Klik Import
    8. Pilih file backup yg di kehendaki atau yang sudah dikirim dari hp lama ke hp baru tadi.
    9. Tunggu prosesnya sampai selesai.
